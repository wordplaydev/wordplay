Στρώσεις εικόνων η μία πάνω στην άλλη

Μερικές φορές θέλουμε να τοποθετήσουμε εικόνες σε στρώσεις για να δημιουργήσουμε διασκεδαστικές σκηνές ή δημιουργικά σχήματα.

Για παράδειγμα, μπορείτε να χρησιμοποιήσετε πολλαπλά επίπεδα στο έργο σας @Stage για να στοιβάζετε και να τακτοποιείτε τις εικόνες όπως θέλετε.

Κοίτα! Ορίστε μια χελώνα που έφτιαξα χρησιμοποιώντας μόνο σχήματα!Δοκιμάστε να αλλάξετε τους αριθμούς για κάθε σχήμα. Μπορείτε να τα μετακινήσετε, να τα αλλάξετε μέγεθος, ακόμη και να τα περιστρέψετε.Παίξε και δες τι μπορείς να δημιουργήσεις!

\¶Σημείωση: Δείτε την γκαλερί για ένα άλλο έργο<παράδειγμα@https://wordplay.dev/project/example-Layers?play>της στρωματοποίησης εικόνων.¶
Stage([
		Shape(Circle(3m 0m 1m 0m))
		Shape(Polygon(2m 3 0m -12m 0m) rotation: -30°)
		Shape(Polygon(7m 6 0m -6m 0m))
		Shape(Rectangle(-11m -9m 11m -12m 10m) rotation: 25°)
		Shape(Rectangle(-11m -9m 11m -12m 10m) rotation: -25°)
	] place: Place(0m 10m -30m))\

Θα μπορούσες να φτιάξεις μερικά πολύ ωραία πράγματα με στρώσεις! Δοκίμασέ το!
