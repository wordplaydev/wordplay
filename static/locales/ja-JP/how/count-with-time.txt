時間が経つにつれて数え上げていく

多くのプロジェクトでは、時間とともに成長していく何かが必要となる。例えば、タイマー、スコア、カウントダウンなどだ。カウンターは、 @Time   @Stream と共に @Reaction 。

A @Reaction 3つの要素から構成される。開始値、監視対象、そしてその対象が変化したときの新しい値である。ここでは、\0\、 時計  @Time そして、時計が進むたびに1ずつ加算していく。

\count: 0 … ∆ Time(1000ms) … count + 1
Phrase(count → '')\

毎秒、 @Time 火災、 @Reaction 通知付き\∆ Time(1000ms)\、 そして \count\になる\count + 1\。の\→ ''\最後に数値をテキストに変換するので @Phrase それを証明できます。

時間だけでなく、任意の入力を使用できます。このカウンターは、Button() が押された回数をカウントします。

\count: 0 … ∆ Button() … count + 1
Phrase(count → '')\

毎回 @Button ストリームはクリックまたはタップで変更され、カウントが1つ増加します。
