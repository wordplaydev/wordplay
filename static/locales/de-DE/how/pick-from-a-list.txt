Wähle zufällig etwas aus einer Liste aus

Möchten Sie, dass Ihr Projekt Sie bei jeder Ausführung überrascht? Erstellen Sie eine Liste mit Optionen und wählen Sie eine davon aus. @Random Die 

@Random kann uns eine Zufallszahl liefern, und eine Liste kann anhand der Zahl nachgeschlagen werden mit\[n]\DieKombiniere sie, um ein zufälliges Emoji auszuwählen:

\emojis: ['🌈' '🌟' '🎈' '🦄' '🍩']
Phrase(emojis.random())\

Sie können auch verwenden @Random mit zwei Zahlen, die eine ganze Zahl zwischen ihnen auswählt.Zum Beispiel, um einen auszuwählen/neu/Jede Sekunde ein zufälliges Emoji, dazu etwas mischen @Time und ein @Reaction :

\emojis: ['🌈' '🌟' '🎈' '🦄' '🍩']
pick: emojis[1] … ∆ Time(1000ms) … emojis[Random(1 emojis.length())]
Phrase(pick)\

Jeder Tick von @Time , Die  @Reaction wählt eine neue Überraschung.
